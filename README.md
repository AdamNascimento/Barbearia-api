# Barbearia System

Sistema de gerenciamento para barbearias com painel administrativo.

## Funcionalidades
- Agendamentos com CRUD completo
- Filtro por data e barbeiro
- Painel de assinantes (planos mensais)
- Cards de resumo do dia

## Tecnologias
- Node.js + Express
- MySQL
- React + Vite + Axios

## Como usar
1. Configure o `.env` com suas credenciais do MySQL
2. Rode o `BD agendamentos.sql` no seu banco
3. `npm install` e `node api.js` na pasta da API
4. `npm install` e `npm run dev` na pasta do painel
